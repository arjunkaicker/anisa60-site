# Studio 60 — Anisa's 60th Birthday

A static site for Anisa's 60th birthday celebration on **Friday, July 24, 2026** at the Westside Arts Center, Atlanta, GA.

**Live site:** https://arjunkaicker.github.io/anisa60-site/

## Pages

- **Home** — hero, countdown, RSVP CTA, section grid
- **RSVP** — form (currently a stub; hook up to Formspree before launch)
- **Run of Show** — entertainment + catering timelines
- **Venue** — Westside Arts Center, map, parking
- **Dress Code** — black with metallic accents, style inspiration
- **Request a Song** — DJ song request form (stub)
- **Travel** — flight, hotel (Hyatt Centric Buckhead), getting around
- **Advocacy** — $60K fundraising goal, giving tiers, charity partners

## Tech

Plain HTML, CSS, vanilla JS. No build step. Photos hot-linked from Unsplash (royalty-free).

## Deploy

Hosted via GitHub Pages on the `main` branch. Custom domain `anisa60.com` to be wired up via DNS once registrar is confirmed.
